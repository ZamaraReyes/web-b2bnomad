Sistema de nomadcoins

Hemos creado un sistema de puntos, NomadCoins (NC), para que cada casa obtenga un valor por noche en función de la ubicación, utilidades, servicios, comodidades, etc.. Este valor no se puede cambiar, de manera que conseguimos un sistema justo y equitativo para todos.

Cada NC tiene un valor de 0,02€. Aún así, ahorrarás con nuestros paquetes.
